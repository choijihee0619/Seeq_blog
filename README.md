# 📚 SeeQ LLM 요약 블로그

## 프로젝트 개요
SeeQ는 스마트 안경 기반 AI 정보 요약·검색 서비스입니다. 
Phase 1에서는 웹 기반 LLM 요약 블로그를 구현합니다.

## 기술 스택
- **Backend**: FastAPI, SQLAlchemy, MySQL
- **AI**: OpenAI GPT-4.1-nano
- **Frontend**: React (향후 구현)
- **Database**: MySQL
- **Deployment**: 향후 결정

## 주요 기능
- 📝 텍스트 입력 시 자동 AI 요약 생성
- 🏷️ 카테고리별 문서 관리
- 🔍 검색 및 필터링
- ✨ 하이라이트 및 키워드 추출
- 📊 문서 통계 및 분석

## 개발 환경 설정

### 1. 저장소 클론
```bash
git clone <repository-url>
cd seeq-blog
```

### 2. Python 가상환경 설정
```bash
python -m venv venv
source venv/bin/activate  # macOS/Linux
# 또는
venv\Scripts\activate     # Windows
```

### 3. 패키지 설치
```bash
pip install -r requirements.txt
```

### 4. 환경변수 설정
`.env` 파일을 생성하고 다음 내용을 입력:
```bash
DATABASE_URL=mysql+pymysql://username:password@localhost:3306/seeq_blog
OPENAI_API_KEY=your_openai_api_key_here
SECRET_KEY=your-secret-key-here
```

### 5. 데이터베이스 설정
```bash
# MySQL 데이터베이스 생성
mysql -u root -p
CREATE DATABASE seeq_blog;
```

### 6. 서버 실행
```bash
cd backend
uvicorn app.main:app --reload --host 0.0.0.0 --port 8000
```

## API 문서
서버 실행 후 다음 URL에서 API 문서 확인:
- Swagger UI: http://localhost:8000/docs
- ReDoc: http://localhost:8000/redoc

# 읽기 플랫폼 웹 인터페이스

## 개요
NotebookLM을 벤치마크로 한 종합적인 읽기 플랫폼으로, 단순한 문서 뷰어를 넘어 **지능형 학습 동반자** 역할을 수행하는 구조로 설계되었습니다.

## 기능 구성

### 1. 홈페이지

#### 주요 기능
- **히어로 섹션**: 서비스 소개 및 시작하기 버튼
- **빠른 액션 카드**: 
 - 카메라로 촬영
 - 파일 업로드  
 - 내 라이브러리
- **최근 활동**: 진행 중인 문서 및 메모 현황 표시
- **반응형 디자인**: 모바일/데스크톱 최적화

### 2. 콘텐츠 입력 페이지

#### 입력 방법
- **카메라 촬영**: 실시간 OCR 텍스트 인식
- **파일 업로드**: PDF, 이미지 드래그앤드롭
- **직접 입력**: 텍스트 타이핑/붙여넣기
- **언어 선택**: 한국어, English, 日本語, 中文
- **미리보기**: 인식된 텍스트 확인 및 수정

### 3. 스피커모드 페이지

#### 텍스트 뷰어
- **반응형 레이아웃**: 텍스트 영역 + 컨트롤 패널
- **하이라이트 도구**: 4가지 색상 선택
- **인라인 메모**: 텍스트 옆 말풍선 형태
- **메모 네비게이션**: 작성한 주석 간 이동

#### TTS 컨트롤
- **재생/정지**: 음성 읽기 제어
- **진행률 표시**: 시각적 진도 바
- **속도 조절**: 0.5x ~ 2.0x 배속
- **음성 선택**: 다양한 TTS 음성

#### 메모 시스템
- **실시간 메모**: 읽기 중 즉석 메모 작성
- **음성 메모**: STT 연동 음성 입력
- **메모 리스트**: 현재 문서의 모든 메모 표시

### 4. 분석 및 요약 페이지

#### AI 분석
- **AI 요약**: 핵심 내용 자동 추출
- **키워드 추출**: 중요 개념 태그화
- **마인드맵**: 개념 간 연관관계 시각화

#### 학습 확장
- **관련 도서**: 연관 추천 도서
- **추천 영상**: 관련 교육 콘텐츠
- **심화 주제**: 더 깊은 학습 방향

#### LLM 챗봇
- **대화형 AI**: 읽은 내용 기반 Q&A
- **빠른 질문**: 6가지 사전 정의 질문 버튼
 - 핵심 메시지 질문
 - 작가 의도 분석  
 - 실생활 적용
 - 작품 비교
 - 퀴즈 생성
 - 토론 주제
- **음성 입력**: STT 연동 음성 질문
- **대화 관리**: 채팅 기록 및 초기화

### 5. 메모 및 독후감 페이지

#### 메모 관리
- **통합 메모 뷰**: 문서별 모든 주석 정리
- **시간순 정렬**: 작성 순서별 메모 표시
- **메모 검색**: 키워드로 과거 메모 찾기

#### 독후감 작성
- **구조화 템플릿**: 
 - 제목, 줄거리, 인상깊은 구절, 느낀점
- **평점 시스템**: 5점 만점 별점 평가
- **하이라이트 연동**: 저장된 구절 자동 가져오기
- **음성-텍스트**: 말로 표현 후 자동 정리
- **AI 도움말**: 독후감 작성 가이드

### 6. 내 라이브러리 페이지

#### 문서 관리
- **카테고리 필터**: 전체, 독서, 학습, 일상, 기타
- **정렬 옵션**: 최근순, 이름순, 진행률순
- **진행률 표시**: 시각적 읽기 진도 바
- **컬러 태그**: 카테고리별 색상 구분

#### 컬렉션
- **독후감 모음**: 작성한 독후감 갤러리
- **즐겨찾는 구절**: 하이라이트 구절 모음
- **학습 통계**: 읽은 책, 읽기 시간, 메모 수, 목표 달성률

### 7. 사용자 설정 페이지

#### 프로필 관리
- **기본 정보**: 이름, 나이, 프로필 사진
- **관심 분야**: 문학, 과학, 철학, 역사 등
- **읽기 목표**: 일일/월간 목표 설정

#### 접근성 설정
- **시각 설정**: 폰트 크기, 고대비 모드, 색상 테마
- **청각 설정**: TTS 음성, 읽기 속도 조절
- **조작 설정**: 음성 명령, 키보드 내비게이션

#### 읽기 환경
- **언어 설정**: 인터페이스 언어, 자동 번역
- **메모 설정**: 자동 저장 간격, 기본 하이라이트 색상
- **알림 설정**: 읽기 알림, 복습 알림, 목표 달성 알림

#### 데이터 및 보안
- **데이터 관리**: 백업/복원, 계정 연동
- **보안 설정**: 비밀번호, 2단계 인증, 개인정보
